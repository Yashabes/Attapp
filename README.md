# Test App Frontend

A React frontend-only test application built with Vite and React Router.

## Files

- `index.html` — Vite entry HTML
- `package.json` — dependencies and scripts
- `vite.config.js` — Vite React plugin configuration
- `src/main.jsx` — React entry point
- `src/App.jsx` — routed React app and page components
- `src/index.css` — application styling

## Routes

- `/` — Home page
- `/login` — Login page
- `/quizzes` — Quiz selection page
- `/quiz/:id` — Quiz detail page
- `/results` — Results page

## Run

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start development server:

   ```bash
   npm run dev
   ```
